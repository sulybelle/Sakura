1. Функционалдылық




Веб-сайт құрылымы мен интерфейсіHTML құжатының дұрыс құрылымы (Doctype, head, body).Семантикалық тегтерді қолдану (header, nav, main, section, article, footer).Беттер арасындағы логикалық навигация (меню, сілтемелер).CSS стильдерінің каскадтылық принципін дұрыс қолдану.

Интерактивтілік (JavaScript)
 Пайдаланушы әрекеттеріне жауап беретін интерактивті элементтер (оқиғаларды өңдеу).DOM элементтерін динамикалық өзгерту (мазмұнды, стильді жаңарту).Формаларды өңдеу және деректерді валидациялау (клиент жағында).

Деректермен өзара әрекеттесу (Back-end)
Пайдаланушыны тіркеу және жүйеге кіру (аутентификация).Деректерді серверге жіберу және мәліметтер базасында сақтау (мысалы, мақалалар, тауарлар, пікірлер).Деректерді серверден алу және экранға шығару.
Қателерді өңдеу 
Формаларды бос жібермеу (клиент және сервер жағында).Серверден келген қателерді дұрыс көрсету (мысалы, "Мұндай пайдаланушы жоқ").Жоқ беттерді (404) өңдеу.

Медиа файлдармен жұмыс
Суреттерді, видео немесе басқа медианы дұрыс көрсету.Медиа файлдарды серверге жүктеу (мысалы, аватар, сурет галереясы).

2. Жұмыстың дұрыстығы




Қатесіз жұмыс (6 ұпай):

Қолданба браузерде қателерсіз жүктеледі.Барлық сілтемелер мен батырмалар жұмыс істейді.Консольде JavaScript қателері жоқ.0 ұпай: Қолданба жүктелмейді немесе жұмыс істеу мүмкін емес, жиі қателер береді.
1 ұпай: Қолданба жүктеледі, бірақ бірден қате береді, негізгі функциялар жұмыс істемейді.2 ұпай: Қолданба жүктеледі, негізгі функциялар жұмыс істейді, бірақ консольде қателер бар және кейбір сілтемелер жұмыс істемейді.
3 ұпай: Қолданба тұрақты, бірақ консольде ескертулер (warning) бар. Кейбір кішігірім функцияларда қате кездеседі.4 ұпай: Қолданба тұрақты жұмыс істейді, консольде қателер жоқ, бірақ кейбір сирек жағдайларда (мысалы, желі баяу болғанда) кішігірім мәселелер туындайды.
5 ұпай: Қолданба өте тұрақты, барлық функциялар жұмыс істейді, консольде қателер жоқ, бірақ әлі де сынаудан өтпеген бірлі-жарым жағдайлар болуы мүмкін.6 ұпай: Қолданба мүлтіксіз жұмыс істейді, барлық жағдайлар тексерілген, ешқандай қате немесе ескерту жоқ.

Пайдаланушыны ақпараттандыру (4 ұпай)
Деректер жүктеліп жатқанда "Жүктелуде..." индикаторы көрсетіледі.Әрекет сәтті орындалғанда хабарлама шығады (мысалы, "Сәтті сақталды").0 ұпай: Ешқандай хабарлама немесе индикатор жоқ, пайдаланушы не болып жатқанын түсінбейді.2 ұпай: Негізгі орындарда хабарламалар мен жүктеу индикаторлары бар, бірақ олар әрдайым дұрыс көрсетіле бермейді (мысалы, жүктеу бітсе де индикатор тұра береді).4 ұпай: Барлық әрекеттер кезінде пайдаланушы толық ақпараттандырылады: не болып жатқаны, не болғаны, қате кетсе не істеу керектігі көрсетіледі.

4. Дизайн және UX




Дизайн эстетикасы 
Қаріптер (3 ұпай):
0 ұпай: Қаріптер тым кішкентай немесе түсі оқуға келмейтіндей, мәтінді оқу мүмкін емес.1 ұпай: Қаріптерді оқуға болады, бірақ стилі дұрыс таңдалмаған (мысалы, әдемі емес немесе түсі фонмен араласып кетеді).2 ұпай: Қаріптер жақсы оқылады, бірақ иерархия дұрыс сақталмаған (мысалы, заголовоктар мен мәтін арасында айырмашылық аз).3 ұпай: Қаріптер өте жақсы оқылады, өлшемдері дұрыс, иерархия сақталған, стильді таңдау сәтті.

Түс схемасы (3 ұпай):
0 ұпай: Түстер көзді ауыртады, мәтін фонға сіңіп кеткен, үйлесім жоқ.1 ұпай: Түстер негізінен сай келеді, бірақ кейбір элементтер (мысалы, батырмалар) жалпы стильден бөлектеніп тұр.2 ұпай: Түс схемасы үйлесімді, барлық элементтер бір стильде, бірақ "түс психологиясы" ескерілмеген (мысалы, қате туралы хабарлама жасыл түспен көрсетілген).3 ұпай: Түс схемасы өте үйлесімді, брендке сай, әр түстің мағынасы дұрыс қолданылған.

Элементтердің орналасуы (4 ұпай):
0 ұпай: Элементтер ретсіз орналасқан, бір-бірінің үстіне шығып тұр.1 ұпай: Элементтер орналасқан, бірақ ара қашықтықтар (margin, padding) дұрыс емес, тығыз не тым шашылыңқы.2 ұпай: Элементтердің көпшілігі логикалық орналасқан, бірақ кейбір блоктар ыңғайсыз жерде тұр.3 ұпай: Орналасуы жақсы, заманауи дизайн принциптері сақталған, кішігірім ұсақ-түйектер ғана түзетуді қажет етеді.4 ұпай: Элементтердің орналасуы өте ыңғайлы әр эстетикалық, F-немесе Z-тәрізді оқу заңдылықтары сақталған.

Интуитивтілік және ыңғайлылық (10 ұпай)
Навигация (3 ұпай):
0 ұпай: Мәзір жоқ немесе оны пайдалану мүмкін емес, қайда басу керектігі белгісіз.1 ұпай: Мәзір бар, бірақ атаулары түсініксіз, пайдаланушы қателеседі.2 ұпай: Навигация түсінікті, бірақ кейбір беттерге жету үшін көп клик жасау керек.3 ұпай: Навигация өте түсінікті, меню логикалық орналасқан, әр бетке 1-2 кликпен жетуге болады.

Маңызды басқару элементтері оңай қол жетімді (3 ұпай):
0 ұпай: Маңызды батырмалар (сақтау, жіберу) жоқ немесе оларды табу мүмкін емес.1 ұпай: Батырмалар бар, бірақ олар өте кішкентай немесе түсі көрінбейді.2 ұпай: Басқару элементтері қол жетімді, бірақ әрқашан көзге түсе бермейді.3 ұпай: Барлық маңызды басқару элементтері үлкен, анық көрінеді және бірден қол жетімді.

Адаптивтілік (4 ұпай):
0 ұпай: Сайт тек компьютерде ғана көрінеді, телефонда бүкіл сайтты кішірейтіп көруге тура келеді.1 ұпай: Сайт телефонда көрінеді, бірақ мәтіндер тым кішкентай, батырмалар басылмайды.2 ұпай: Сайт барлық құрылғыларда ашылады, бірақ кейбір элементтер экранға сыймай қалады немесе жылжып кетеді.3 ұпай: Сайт барлық құрылғыларға бейімделген, бірақ планшетте кейбір блоктар тым кең не тым тар болып көрінеді.4 ұпай: Сайт барлық құрылғыларда (телефон, планшет, ноутбук, 4K монитор) тамаша көрінеді, барлық элементтер дұрыс орналасқан.

5.  Код сапасы (10 ұпай)




1.Кодтың тазалығы мен оқылуы (3 ұпай)
Айнымалылар мен функциялардың атаулары түсінікті (ағылшын тілінде).Код бір стильде жазылған (форматтау).Артық, комментарийге айналдырылған код жоқ.
0 ұпай: Кодты оқу мүмкін емес, айнымалы атаулары a, b, c деп берілген, тармақтар (indentation) жоқ.1 ұпай: Кодты түсіну қиын, бірақ кейбір жерлерде атаулар түсінікті.2 ұпай: Код негізінен оқылады, айнымалы атаулары жақсы, бірақ кейбір функциялар тым ұзын немесе түсініктемелер жоқ.3 ұпай: Код өте таза, айнымалы атаулары ағылшынша және мағынасы анық, біркелкі стиль сақталған.

2. Құрылымдылығы (4 ұпай)
HTML, CSS, JavaScript (React) компоненттерге бөлінген.Файлдар мен қалталар құрылымы логикалық (мысалы, components, pages, utils).Қайталанатын код функцияларға немесе компоненттерге бөлінген.
0 ұпай: Барлық код бір файлда (мысалы, бір гигант index.html ішінде).1 ұпай: Файлдарға бөлінген, бірақ құрылымы жоқ, компоненттер бір-біріне шатасып кеткен.2 ұпай: Негізгі бөліну бар (HTML, CSS, JS бөлек), бірақ React компоненттері дұрыс ұйымдастырылмаған.3 ұпай: Құрылымы логикалық, қайталанатын код бөліктерге бөлінген, бірақ кейбір файлдар әлі де үлкен.4 ұпай: Құрылымы өте жақсы, әр компонент өз файлында, қалталар дұрыс аталған, кодты кеңейту оңай.

3. Қателерді өңдеу (3 ұпай)
try...catch блоктары қолданылған (серверде және клиентте).Потенциалды қауіпті кодтар тексерілген.
0 ұпай: try...catch мүлдем қолданылмаған, қате кетсе сайт толығымен құлайды.1 ұпай: try...catch бар, бірақ тек бір-екі жерде ғана, көп жерде қауіпсіздік қарастырылмаған.2 ұпай: Негізгі қауіпті жерлерде (мысалы, серверге сұрау) қателер өңделген, бірақ кейбір жағдайлар ескерілмеген.3 ұпай: Барлық ықтимал қателер өңделген, сайт ешқашан күтпеген жерден құламайды, пайдаланушыға дұрыс хабарлама шығады.

6. Технологияны қолдану




1. React (8 ұпай)
 Функционалды компоненттер дұрыс қолданылған (2 ұпай)
0 ұпай: Класс компоненттері ғана қолданылған немесе компоненттер дұрыс емес1 ұпай: Функционалды компоненттер бар, бірақ аралас қолданылған2 ұпай: Тек функционалды компоненттер қолданылған, дұрыс құрылған

Props арқылы деректер берілген (2 ұпай) 
0 ұпай:  Props қолданылмаған, деректер тікелей компоненттерде.1 ұпай:  Props қолданылған, бірақ деректер ағыны шатастырылған.2 ұпай: Props дұрыс қолданылған, деректер ата-анадан балаға беріледі.

State (күй) дұрыс басқарылған (2 ұпай)
0 ұпай:  State мүлдем жоқ немесе дұрыс емес жерде қолданылған.1 ұпай:  useState қолданылған, бірақ күй тым көп немесе шатасып кеткен.2 ұпай: Күй дұрыс басқарылған, әр компоненттің өз күйі бар, қажетсіз рендерлер жоқ.

React Router қолданылған (беттер арасында ауысу) (2 ұпай)
0 ұпай:  Маршрутизация жоқ, барлығы бір бетте.1 ұпай:  React Router орнатылған, бірақ кейбір сілтемелер жұмыс істемейді.2 ұпай: React Router дұрыс қолданылған, барлық беттерге өту мүмкін.

2. Node.js және Дерекқор (7 ұпай)
REST API құрылған (3 ұпай)
0 ұпай : API жоқ1 ұпай: API бар, бірақ тек бір-екі эндпоинт жұмыс істейді2 ұпай: API бар, негізгі эндпоинттер жұмыс істейді (GET, POST), бірақ PUT, DELETE жоқ3 ұпай: Толық REST API (GET, POST, PUT, DELETE) дұрыс жұмыс істейді

Құпия сөздер хэштелген (мысалы, bcrypt) (2 ұпай)0 ұпай: Құпия сөздер ашық түрде сақталған.1 ұпай: Хэштеу бар, бірақ әлсіз алгоритм қолданылған.2 ұпай: Құпия сөздер дұрыс хэштелген (bcrypt және т.б.).

Жалпы құралдар (5 ұпай)

7. Тестілеу




Қолмен тестілеу (5 ұпай) 
Барлық негізгі функциялар қолмен тексерілген.Әртүрлі браузерлерде тексерілген (Chrome, Firefox).API құралдарымен тексерілген (мысалы, Postman).
• 0 ұпай: Ешқандай тестілеу жүргізілмеген.• 1 ұпай: Тек бір ғана функция тексерілген (мысалы, тек кіру беті).• 2 ұпай: Негізгі функциялар тексерілген, бірақ жиі қолданылмайтын жерлерде қателер қалуы мүмкін.• 3 ұпай: Барлық функциялар тексерілген, бірақ тек бір браузерде ғана (мысалы, тек Chrome).• 4 ұпай: Барлық функциялар бірнеше браузерде тексерілген, API Postman арқылы тексерілген.• 5 ұпай: Функционалдық тестілеу толық жүргізілген, шекті мәндер тексерілген, барлық браузерлер мен құрылғыларда сыналған.

8. Құжаттама




• 0 ұпай: Құжаттама мүлдем жоқ.• 1 ұпай: Тек бір файл бар (мысалы, README.md), бірақ ішінде тек жоба аты ғана жазылған.• 2 ұпай: README.md файлы бар, онда жобаның аты, автор және қысқаша сипаттамасы жазылған.• 3 ұпай: README.md файлында толық нұсқаулық бар: қалай орнату керек, қандай технологиялар қолданылған.• 4 ұпай: README.md-тен басқа, код ішінде маңызды жерлерге түсініктемелер (комментарийлер) жазылған.• 5 ұпай: Толық құжаттама: README.md, API документациясы (негізгі эндпоинттер сипатталған), пайдалану нұсқаулығы немесе скриншоттар қосылған.
